## YINXUN IE Mode troubleshooting flow

When YINXUN cannot open:

1. Check Edge default browser settings:
   - `edge://settings/defaultBrowser`
   - confirm the YINXUN URL is set to open in IE Mode.
2. Check whether the URL should use `http` or `https`.
3. If it still cannot open, re-add the YINXUN URL to IE Mode.
4. If it still cannot open, clear cookies/site data for the system.
5. If a password prompt appears, ask the user instead of assuming the password.
