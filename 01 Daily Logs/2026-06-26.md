## Troubleshooting: meeting room cannot share screen

Problem:

- 會議室 is already on HDMI2, but screen sharing still does not work.

Possible fix observed:

1. Unplug rear HDMI first.
2. Change HDMI option first, for example to HDMI 3.
3. Plug the antenna/controller into the user's device.
4. Plug rear HDMI again.
5. Test screen sharing.

See [[Meeting Room Display and HDMI]].

## Troubleshooting: printer

First fix:

1. Open Windows Services.
2. Find Print Spooler.
3. Stop it.
4. Start it again.
5. Test print.

See [[Printer Spooler]].
