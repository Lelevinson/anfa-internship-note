## Cleaned rough meaning

- In the meeting room / 會議室, some customer laptops have USB Type-C ports that do not support the wireless screen-sharing dongle/device.
- Clue: the TV already shows the standby screen, but the laptop still does not share anything.
- In that case, usually no need to repeatedly unplug/replug the meeting-room device.
- Unplug/replug can be used only to show/confirm the error that the Type-C port is not supported, but usually it is unnecessary.
- If the TV is not on the standby screen, try unplugging and plugging back in.
- If the Type-C port is not supported, use the table HDMI cable instead.
