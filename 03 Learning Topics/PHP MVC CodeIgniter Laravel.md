## Simple definitions

- PHP: backend language commonly used for web systems.
- MVC: pattern that separates an app into Model, View, and Controller.
- CodeIgniter: PHP framework, often used for simpler or older business systems.
- Laravel: PHP framework with a larger ecosystem and more built-in structure.

## MVC mental model

```text
Controller = receives request and coordinates logic
Model = talks to data/database
View = displays page/output
```
