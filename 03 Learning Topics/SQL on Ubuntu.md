# SQL on Ubuntu

tags: #internship #learning #sql #ubuntu #database

## Why this matters

The raw notes mention running SQL scripts in Ubuntu because running large SQL scripts in a GUI can lag a lot.

## Notes

- `pv` can be used to monitor progress when importing large SQL files.
- GUI database tools can freeze or lag with large scripts.
- The database may need settings changed to allow other devices/users to connect.

## Things to learn

- How to import SQL script from terminal
- How to check database service status
- How to configure bind address / allowed host
- How to check database user permissions
- How to verify port connectivity
