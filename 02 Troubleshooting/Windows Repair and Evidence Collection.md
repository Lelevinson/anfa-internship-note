# Windows Repair and Evidence Collection

tags: #internship #troubleshooting #windows #support

## When this applies

Use this when a Windows PC has a system-level problem and you need to either try a safe command or collect evidence before escalating.

## Possible repair command

```powershell
sfc /scan now
```

Note: this may require admin permission.

## If you cannot fix directly

Record complete information:

- photo or screenshot
- exact error message
- PC/user affected
- what the user was trying to do
- when it happened
- whether it happens repeatedly
- what was already tried

## Why this matters

For IT support, good escalation info is often more useful than guessing. It lets your mentor or senior reproduce the problem faster.
