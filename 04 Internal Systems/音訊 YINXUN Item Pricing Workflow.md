# 音訊 YINXUN Item Pricing Workflow

tags: #internship #internal-system #yinxun #task

## When this applies

Use this for the task where Connie sends an Excel file and the needed values must be checked from YINXUN.

## Excel task scope

Only work on:

- Sheet 2
- Sheet 3

Columns to fill:

- 售價
- OPD

## YINXUN path

```text
音訊 / YINXUN
→ 管理部
→ 檢驗與藥品項目
→ search NAME
→ double click
→ 賈母表項目
```

## Fields to copy/fill

Fill into the Excel:

- 售價
- OPD
- 買購售, if the field exists / is present

## Step-by-step

1. Open YINXUN.
2. Go to 管理部.
3. Open 檢驗與藥品項目.
4. Search using the item name from Excel.
5. Double click the matching item.
6. Open/check 賈母表項目.
7. Copy/fill 售價 and OPD.
8. Also fill 買購售 if present.
9. Repeat for Sheet 2 and Sheet 3 only.

## Things to be careful about

- Match the item name carefully.
- Do not fill other sheets unless Connie says so.
- If there are multiple search results, confirm before choosing.
- If a field is missing, mark it for later confirmation instead of guessing.
