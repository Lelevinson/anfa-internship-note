# Raw Note Capture Template

tags: #internship #workflow #template

Use this when typing fast notes during mentor explanations.

## Simple format

```text
MM/DD
Q: question I need to ask
N: thing I learned / thing to study
S: solution / troubleshooting step
T: task I need to do
Problem: unresolved issue
```

## Better raw-note pattern

When possible, include these parts:

```text
System:
Problem / task:
Where:
Steps shown:
Important fields/buttons:
Who to ask:
Need to confirm:
```

## Examples

### Troubleshooting

```text
07/xx
S: printer cannot print. Windows Services > Print Spooler > Stop > Start > test print.
```

### Task

```text
07/xx
T: Connie Excel. Only Sheet 2 and Sheet 3. Fill 售價 and OPD from YINXUN.
```

### Equipment

```text
07/xx
N: CD player in cabinet. ASUS black box on shelf 2. ASUS USB-C cable on shelf 1.
```

## Important

Messy notes are okay. The goal during explanation is capture, not perfect writing. Cleanup can happen later.
